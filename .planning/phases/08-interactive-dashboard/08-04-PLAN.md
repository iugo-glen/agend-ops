  <name>Task 1: Configure Coolify deployment and Google OAuth</name>
  <files>none</files>
  <action>
Glen needs to configure Coolify and Google OAuth for production deployment. Claude cannot access the Coolify UI or Google Cloud Console directly.

**Coolify Setup (per D-06, D-09):**
1. Open Coolify dashboard on home server
2. Create new Application: source = Git repository (this repo), branch = master
3. Set **Base Directory** to `/dashboard` (Coolify clones full repo, builds from this subdir)
4. Set **Build Pack** to "Dockerfile" (auto-detects dashboard/Dockerfile)
5. Set **Port** to 3000
6. Optionally set **Watch Paths** to `dashboard/**` (prevents redeployment on data file changes)
7. Configure **Persistent Storage** as a bind mount:
   - Source (host path): `/Users/glenr/work/todo-list/data` (or wherever the repo is on the Coolify host)
   - Destination (container path): `/data`
   - This gives read access to all NDJSON files AND write access to data/queue/
8. Add **Environment Variables**:
   - `AUTH_SECRET` = generate with `openssl rand -base64 32`
   - `GOOGLE_CLIENT_ID` = from Google Cloud Console (existing or new OAuth client)
   - `GOOGLE_CLIENT_SECRET` = from same OAuth client
   - `NEXTAUTH_URL` = `https://ops.agendsystems.com` (or chosen subdomain per D-09)
   - `DATA_DIR` = `/data`
9. Configure **custom domain** in Coolify (e.g., ops.agendsystems.com per D-09)
10. Deploy

**Google OAuth Setup (per D-08):**
1. Go to Google Cloud Console > APIs and Services > Credentials
2. Either reuse existing OAuth client or create a new one (recommended: new client for dashboard -- it only needs openid/email/profile scopes, not Gmail/Drive)
3. Add to **Authorized redirect URIs**:
   - `https://ops.agendsystems.com/api/auth/callback/google` (production)
   - `http://localhost:3000/api/auth/callback/google` (development -- if not already added)
4. Add to **Authorized JavaScript origins**:
   - `https://ops.agendsystems.com`
   - `http://localhost:3000`
5. If using Internal OAuth consent type: add the subdomain's parent domain (e.g., agendsystems.com) to Authorized Domains
6. Save and wait 5 minutes for propagation

**DNS Setup (per D-09):**
1. Add A record or CNAME pointing the chosen subdomain to the home server's IP
2. If behind Cloudflare: set DNS-only (grey cloud) so Coolify handles TLS via Let's Encrypt
